# Trademark Registration Guide — eooo / Orkestr

> Practical steps for filing US trademarks for the eooo.ai brand.

---

## Marks to File

| Mark | Description |
|------|-------------|
| **eooo** | Company/brand name (eooo.ai) |
| **Orkestr** | Product name (agent orchestration platform) |

## Classes

Both marks filed in:
- **Class 9** — Downloadable computer software
- **Class 42** — Software as a service / technology services

Total: 2 marks x 2 classes = 4 class filings

## Filing Options

### Option A: DIY via USPTO TEAS Plus ($1,000 total)

1. Go to [trademark.gov](https://www.uspto.gov/trademarks)
2. Use **TEAS Plus** filing — $250 per class
3. Total: 4 x $250 = **$1,000**
4. Requirements:
   - Applicant name and address (the LLC, or personal name if filing before formation)
   - Description of goods/services per class
   - Specimen showing the mark in use (screenshot of landing page, product UI)
   - Filing basis: **1(a) "use in commerce"** if already public, or **1(b) "intent to use"** if pre-launch (adds $100 Statement of Use fee later)

### Option B: Use a service ($1,200-1,400 total)

- **Trademarkia** — ~$200 service fee + USPTO fees
- **LegalZoom** — ~$200-400 service fee + USPTO fees
- They handle paperwork, description wording, filing — you review and sign

## Process Timeline

```
File application
    | (~3 months)
USPTO examining attorney reviews
    |
Either approves -> published for opposition (30 days)
    |
Or issues Office Action -> you respond (fix description, etc.)
    |
No opposition -> registration certificate issued
    | (~8-12 months total)
Registered trademark
```

## Before Filing: Search for Conflicts

1. Go to [TESS](https://tess2.uspto.gov/) (USPTO trademark search)
2. Search for "eooo" and "orkestr" to check for conflicts
3. Also search phonetic equivalents ("echo", "orchestr")
4. If something similar exists in Class 9/42, may face an objection

## Usage Rules

- **™** — can use immediately without registration (means "I claim this as my trademark")
- **®** — only after USPTO grants registration (using it before is illegal)
- File under the **LLC name** once formed, not personal name — saves reassignment later
